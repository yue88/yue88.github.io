Retro Revival

INTRODUCTION :

Retro Revival est un site web réalisé de A à Z dans le cadre d'un projet d'anglais.
Objectif : concevoir une boutique en ligne capable de satisfaire les exigences précises d'un client cible.
Notre commanditaire ? Une entreprise fictive imaginée par un autre groupe d'élèves, désireux de lancer leur e-commerce d'objets rétro.

EXECUTION :

  • Conception UI/UX
    • Direction artistique néon-pixel inspirée du synth-wave : palette violet/magenta, police “Press Start 2P”, icônes 8-bit.
    • Parcours utilisateur simplifié : page d’accueil → catégories par décennie → fiches produit → tunnel d’authentification (login/register).
  • Implémentation technique
    • Front-end pur : HTML5 sémantique + CSS Grid/Flexbox, aucune dépendance JS lourde.
    • Assets modulaires : images décoratives injectées en pseudo-éléments pour conserver un site léger.
  • Exigences pédagogiques
    • Collaboration inter-groupes : un « client » (autre équipe) fournit scénario et besoins ; notre équipe livre wireframes, maquettes puis code final.
